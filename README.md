# Messenger
описание

### Локальный запуск

1) Клонируем репозиторий: git clone https://github.com/IlyaOdintsov/messenger

2) Запускаем frontend часть
```

cd client

npm install

npm run dev

```

3) Запускаем backend часть
```

cd server

npm install

npm run dev

```

4) Запускаем redis
```

docker-compose up -d

```
